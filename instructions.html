Welcome to WheelEase!

Your guide to using the platform efficiently.

User Emergency Nurse/Doctor/Employee

User

1. After opening the website, click on Get Started.
2. Click on the USER icon displayed.
3. a. If you are a new user, click on Sign Up and provide the necessary details to register.
   b. If you are already registered, login to the website.

N.B. - You can also use Social Media Links (Google/Facebook/Apple) to login/signup.

4. Select your hospital where you are present.
5. You will see various options, including:
   - Hospital map.
   - Advertisements.
   - Booking section for wheelchairs and stretchers.

6. Select one of the options to view the hospital map, showing:
   - Your current location.
   - Total number of wheelchairs and stretchers available.
   - Service options:
     a) Self-service.
     b) With help staff.

7. Book the service by paying the necessary fees.
8. Receive a confirmation message with a code (for help staff if selected).
9. Click on the Profile Picture to edit personal information if needed.

Emergency

1. After opening the website, click on Get Started.
2. Click on the EMERGENCY icon displayed.
3. You will see three icons:
   - Ambulance
   - Wheelchair
   - Stretcher

4. Click the relevant icon to book the service nearest to you.

Nurse/Doctor/Employee

1. After opening the website, click on Get Started.
2. Click on the USER icon displayed.
3. Enter your Employee ID and password.
4. You will see options for booking:
   - Wheelchair
   - Stretcher
   - Ambulance

5. Select the desired option to view the hospital map and available resources:
   a) Self-service.
   b) With help staff.

6. Receive a confirmation message with a code (for help staff if selected).
7. Click on the Profile Picture to edit personal information or view other employees' details.
